Introduction

The following document explores increasing population size in nodes versus increasing the number of nodes in the network.

Parameters

The parameters for these runs:

From 20 to 1500 chickens

From 10 to 750 farms

1000 simulations per parameter set.

\(\beta\) = 1.44456 Contact rate

\(\gamma\) = 0.167 Recovery rate

\(\mu\) = 0 Base mortality rate

\(\alpha\) = 0.4 Disease mortality rate

\(\phi\) = 0 Infectiousness of environmental virions

\(\eta\) = 0 Degradation of environmental virions

\(\nu\) = 0 Update rate of environmental virions

\(\sigma\) = 0 Virion shedding rate

\(\omega\) = 0.03 Inter-patch movement rate

\(\rho\) = 0.85256 Contact non-linearity parameter

\(\lambda\) = 0 External force of infection

\(\tau\)-crit = 5 Critical surveillance time

\(I\)-crit = 1 Threshold for reporting

\(\pi\)-report = 0.9 Reporting probability

\(\pi\)-detect = 0.9 Detection probability

\(t_c\)-detect = 5 Mean Time in Days until Culling
